현재 메인 화면 — "뚝 떨어지는" 원인 분석

첫 진입 시 사용자가 느끼는 문제점을 분류했습니다. 가장 공감되는 것을 선택해주세요.

A

로딩 → 콘텐츠 전환이 갑작스러움

스피너만 돌다가 콘텐츠가 갑자기 나타남. 스켈레톤이나 페이드인이 없어서 화면이 "툭" 튀는 느낌. 로딩 중에 레이아웃 공간이 예약되지 않아 화면이 점프함.

B

헤더/앱바가 너무 비어있음

AppBar에 "기란장터" 텍스트 + 검색 아이콘만 있고, 검색바/환영 문구/컨텍스트 정보가 없어서 화면 상단이 허전함. 사용자가 "여기서 뭘 해야 하지?" 감을 잡기 어려움.

C

필터만 있고 안내/요약 정보가 없음

서버 필터 칩 아래 바로 카드 리스트가 시작됨. "총 N개 매물", "최근 등록", "인기 매물" 같은 컨텍스트가 없어서 콘텐츠가 맥락 없이 나열되는 느낌.

D

PC에서 화면이 부자연스럽게 늘어남

단일 컬럼 레이아웃이 데스크탑 전체 너비로 늘어남. 하단 네비게이션바가 PC에서 어색함. 콘텐츠 최대 너비 제한이 없어서 카드가 지나치게 넓어짐.

E

에러/빈 상태가 불친절함

네트워크 오류 시 조용히 실패 (에러 UI 없음). 빈 상태 메시지가 항상 동일. 검색 결과 없음 / 필터 결과 없음 구분이 없어서 사용자가 원인을 모름.

F

검색이 숨겨져 있고 필터가 부족함

검색이 모달 다이얼로그 뒤에 숨겨져 있어 발견하기 어려움. 카테고리 필터/정렬 옵션이 없어서 원하는 매물을 찾기 위한 도구가 부족함.

복수 선택 가능합니다. 가장 "뚝 떨어지는" 느낌에 기여하는 항목들을 골라주세요.
