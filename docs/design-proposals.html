Proposal 01

Proposal 02

Typography System

한국어 가독성을 최우선으로, 판타지 감성을 더한 타이포그래피 시스템. Black Han Sans(제목)와 Noto Sans KR(본문)의 조합으로 게임다운 느낌과 정보 전달력을 동시에 확보합니다. Noto Serif KR은 아이템 설명이나 특수 섹션에 중세적 분위기를 더합니다.

Navigation & Header

Responsive Breakpoints
Desktop (1200px+) 3-column: sidebar + grid(3col) + aside
Tablet (768px-1199px) 2-column: grid(2col) + collapsible sidebar
Mobile (<768px) 1-column: full-width cards, bottom nav
Flutter Implementation
// Use LayoutBuilder for responsive layout
LayoutBuilder(builder: (context, constraints) {
  if (constraints.maxWidth >= 1200) {
    return Row(children: [
      SizedBox(width: 200, child: CategorySidebar()),
      Expanded(child: ItemGrid(crossAxisCount: 3)),
      SizedBox(width: 240, child: TrendingSidebar()),
    ]);
  } else if (constraints.maxWidth >= 768) {
    return Row(children: [
      Expanded(child: ItemGrid(crossAxisCount: 2)),
    ]);
  }
  return ItemGrid(crossAxisCount: 1);
})
